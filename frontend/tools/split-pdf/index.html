What Does Splitting a PDF Mean?

Splitting a PDF divides one multi-page document into multiple smaller PDFs. Useful for sharing specific chapters, extracting invoice pages, separating scanned documents, or breaking up presentations into individual slides.

Lossless PDF Splitting — What Does It Mean?

Most online PDF splitters re-encode (re-compress) page streams when writing output files, degrading image quality. IshuTools uses pikepdf with recompress_flate=False, which copies every byte of every stream exactly as-is. Output files are bit-perfect extracts of the original PDF.

Supported Page Range Syntax

The range field accepts: 1-5 (range), 8 (single page), 1-5, 8, 12-end (mixed), odd, even, first 10, last 5. All ranges are 1-indexed.
