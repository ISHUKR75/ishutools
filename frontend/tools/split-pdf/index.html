Complete Guide

How to Split a PDF: The Complete 2025 Guide

By Ishu Kumar • Updated June 2026

What does "splitting a PDF" mean?

Splitting a PDF means dividing one PDF document into multiple smaller PDF files. Each output file contains a subset of the original pages — useful for extracting a chapter, sharing only relevant sections, reducing file size, or reorganizing scanned documents.

Method 1: Extract a Page Range

The most common need — extracting specific pages. In Page Range mode, type ranges like 5-12, click individual pages in the visual grid, or use Quick Select (All, Odd, Even, First N). The output is one PDF with only those pages.

Method 2: Split Every N Pages

Create equal-size chunks. Set N to 10 and a 100-page PDF becomes 10 files of 10 pages each. The last chunk may be smaller if pages aren't evenly divisible.

Method 3: Split by Bookmarks (Chapters)

If your PDF has a table of contents, this is the smartest approach. Each top-level bookmark entry becomes a separate PDF file — named after the bookmark title. Ideal for textbooks, reports, and structured documents.

Method 4: Blank Page Separators

When scanning physical documents, insert a blank page between each document. This mode auto-detects blank pages and splits there. Each section between blanks becomes one file — great for batches of scanned invoices, forms, or letters.

Method 5: Split by File Size

Group pages until the target file size is reached, then start a new part. Great for sharing large PDFs through size-limited channels like email or messaging apps.

Method 6: Odd and Even Pages

Splits a PDF into two files: odd-numbered pages (fronts) and even-numbered pages (backs). Essential for interleaving duplex scans.

Tips for Best Results

Use the visual page grid for precise selection — click pages to select, shift+click for ranges. Check the live preview chip to confirm the output before splitting. For password-protected PDFs, expand Advanced Options first.
