What is PDF Splitting?

Splitting a PDF means dividing one PDF file into multiple smaller PDF files. You might need to split a PDF to extract specific pages, separate chapters, reduce file size, or share only part of a document. IshuTools Split PDF at ishutools.fun — created by Ishu Kumar (ISHUKR41) — makes this instant and free.

All 7 Split Modes — When to Use Each

1. Every Page — Best for separating scanned documents, creating individual page archives, or distributing pages individually. A 50-page PDF → 50 individual PDFs in a ZIP.

2. Page Ranges — Most flexible mode. Use the visual page grid to click the exact pages you need, or type ranges like 1-5, 8, 12-15. Best for extracting specific sections, chapters, or non-consecutive pages.

3. Every N Pages — Creates equal-size chunks. Set N=10 → every 10 pages becomes one file. Best for creating handouts, study guides, or distributing workbooks in parts.

4. By Chapters/Bookmarks — Reads the PDF's embedded bookmark structure and splits at each chapter. Best for textbooks, eBooks, legal documents, and technical manuals with proper bookmark structure.

5. Odd / Even Pages — Creates exactly 2 files: odd pages (1,3,5…) and even pages (2,4,6…). Best for duplex scanning reassembly, reverse-scan correction, and double-sided print preparation.

6. By File Size — Limits each output file to a maximum MB. Best for email attachments (Gmail 25MB, Outlook 20MB), WhatsApp/Telegram limits, or cloud storage quotas.

7. At Blank Pages — Auto-detects blank separator pages and uses them as split points. Best for batch-scanned physical document archives where blank sheets were inserted to separate documents.

How to Use the Visual Page Grid

IshuTools' unique visual page grid shows every page of your PDF as a numbered, clickable cell. Click any cell to select/deselect it. Shift+click to select a range of pages instantly. Use the quick-select bar for All, None, Odd, Even, First N, or Last N pages. The live split preview chip shows exactly what will be created before you click Split.
