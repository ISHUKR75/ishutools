How to Compress Scanned PDF — Ishu Kumar's Specialist Guide

Scanned PDFs are 5–20× larger than text PDFs because every page is a raw image. Ishu Kumar designed specific handling for scanned documents — here's how to get the best results.

The Problem with Scanned PDFs

  • A 10-page scanned document can be 50–200 MB
  • Each page = a JPEG/PNG image at 300–600 DPI (scanner default)
  • Normal text compressors can't compress image-only pages
  • Most free tools reject files over 10 MB — Ishu's tool has no limit
  • Result: WhatsApp won't send it, email rejects it, govt portals block it

Ishu Kumar's Solution

  • PyMuPDF detects image-only pages automatically
  • Resamples from 600 DPI → 150–200 DPI (High preset)
  • JPEG quality set to 85 — invisible quality difference, huge size saving
  • A 100 MB scanned PDF → 8–15 MB in High mode
  • Or use Screen preset: → 2–5 MB for WhatsApp/email sharing
  • Target Mode: set exactly 500 KB for govt portal upload
