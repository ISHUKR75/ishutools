How PDF Compression Works — Technical Details

PDF files contain streams of compressed data (images, fonts, page content) plus a cross-reference table of object locations. IshuTools compresses at multiple levels:

  • Image streams — JPEG re-encoding at controlled quality + DPI downsampling
  • Font streams — subsetting to only embed used glyphs
  • Content streams — DEFLATE-9 recompression of page drawing commands
  • Object streams — merging multiple objects into compressed ObjStm containers
  • Dead objects — removing orphan objects not referenced by the page tree
  • Duplicate images — MD5-based deduplication of identical embedded images
  • Metadata — stripping XMP and DocInfo entries that add size with no visual benefit
