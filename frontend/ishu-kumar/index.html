About the Creator

Ishu Kumar — ISHUKR41

Ishu Kumar (GitHub: ISHUKR41 / ISHUKR75) is an Indian software developer who single-handedly designed, built and maintains IshuTools.fun — a complete suite of 36+ professional-grade free PDF tools used by thousands of people worldwide.

Who is Ishu Kumar?

Ishu Kumar is an Indian developer passionate about building free, accessible web tools that solve real problems. He created IshuTools.fun (also known as Ishu Tools, or simply "Ishu PDF Tools") because he was frustrated with paid PDF tools that added watermarks, required signups, or charged for basic features. His mission: make professional PDF processing free for everyone.

Known online as ISHUKR41 and ISHUKR75 on GitHub, Ishu Kumar has built every single tool at IshuTools.fun himself — from the Python Flask backend to the pixel-perfect frontend. All 36+ tools are entirely his work, 100% free, with no watermarks, no signups, and no data collection.

What is IshuTools.fun?

IshuTools.fun is Ishu Kumar's flagship project — a professional PDF tools suite with tools spanning every PDF workflow:

  • Organize: Merge PDF, Split PDF, Compress PDF, Remove Pages, Extract Pages, Rotate
  • Convert to PDF: JPG to PDF, Word to PDF, Excel to PDF, PowerPoint to PDF, HTML to PDF
  • Convert from PDF: PDF to JPG, PDF to Word, PDF to Excel, PDF to PowerPoint, PDF to PDF/A
  • Edit PDF: Add Page Numbers, Add Watermark, Crop PDF, Edit PDF
  • Security: Unlock PDF, Protect PDF, Sign PDF, Redact PDF, Compare PDF
  • AI Tools: AI Summarize PDF, Translate PDF, OCR PDF (Tesseract)

Ishu Kumar's Philosophy

Ishu Kumar built IshuTools.fun with a clear philosophy: professional tools must be free. Every tool at IshuTools.fun is genuinely free — no freemium trap, no "upgrade to remove watermark", no email required, no credit card needed. Files are processed 100% privately on the server and immediately deleted after you download.

Ishu Kumar on IshuTools.fun:
"I built this because I believe professional tools should be free for everyone — students in India, freelancers anywhere, anyone who needs to work with PDFs. No watermarks. No signups. No excuses." — Ishu Kumar (ISHUKR41)

Technology Stack

Ishu Kumar built IshuTools.fun using a fully Python + JavaScript stack. Backend: Python · Flask · pypdf · pikepdf · PyMuPDF · ReportLab · Pillow · img2pdf · Tesseract OCR · pdf2docx · openpyxl · python-pptx · deep-translator · Gunicorn. Frontend: Vanilla HTML · CSS · JavaScript with GSAP animations, Sortable.js, Chart.js, Typed.js, and a canvas particle system. Served via Gunicorn with gthread workers for concurrency.

Journey

The Story of Ishu Kumar & IshuTools.fun

How an Indian developer built India's most comprehensive free PDF tools suite — from scratch, alone, 100% free.

The Problem — Why Ishu Kumar Built This
Tired of Paid, Watermarked PDF Tools
Ishu Kumar (ISHUKR41) was fed up with PDF tools that charge money for basic features, add watermarks to your documents, require you to create an account, or limit file sizes. As a student and developer in India, he knew that professional tools should be accessible to everyone — free.
2024 — Building IshuTools.fun
Ishu Kumar Builds the First Tools
Ishu Kumar started building IshuTools.fun with Python and Flask. He chose pikepdf and pypdf for lossless PDF processing — no quality loss, no re-encoding. The first tools were Merge PDF, Split PDF, and Compress PDF. Each tool was hand-coded, hand-tested, hand-deployed by Ishu Kumar alone.
2025 — Expansion to 36+ Tools
Ishu Kumar Expands to 36+ Tools
IshuTools.fun grew to 36+ tools covering every PDF workflow — OCR with Tesseract, AI Summarization, Translation in 50+ languages, Office format conversion, PDF security, digital signatures, and more. Ishu Kumar added enterprise-grade features: page ranges, drag-reorder, bookmarks, TOC generation, and password-protected PDF merging.
2026 — v9.0 Enterprise — Today
ishutools.fun v9.0 — Thousands of Users
IshuTools.fun v9.0 is the most advanced free PDF merger available online. Built by Ishu Kumar (ISHUKR41), it features enterprise merge logic, smart postprocessing, quality scoring, compression optimization, SSE real-time progress, and a professional UI with animations. Used by 51,000+ people worldwide. Still 100% free. Still built alone by Ishu Kumar.
